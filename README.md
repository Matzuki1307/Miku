# MikuAssistant

Asistente local para Linux construido con Python y Ollama.

## Características

- Conversación mediante LLM local
- Clasificación de intenciones
- Apertura de aplicaciones
- Integración con Firefox
- Integración con Steam
- Consulta de hora

## Tecnologías

- Python
- Ollama
- Llama 3.2
- EndeavourOS
